# 🛒 QuickCart

QuickCart is a cross-platform **React Native e-commerce mobile app** that allows users to browse products, add items to cart, place orders, and receive notifications.

---

## 🚀 Features

- 📦 Product Listing
- 🛒 Add to Cart
- 📑 Order Placement
- 🔔 Firebase Push Notifications
- 📱 Cross-platform (Android & iOS)

---

## 🛠️ Tech Stack

- React Native
- Firebase (Notifications)
- JavaScript / TypeScript

---
